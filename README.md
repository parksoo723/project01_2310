# 쥐 실험 게임
과거: 약초
현재: 실험 도구, 약물 등

## 📌 프로젝트 목적
- 플레이어가 구체적인 실험을 통해 목표를 달하게 하는 것

## 🧠 사용 기술
- Unity Hup
- C#
- ClipStudio
- ibis paint X

## ⚙ 구현 내용
- 게임은 총 30일동안 진행
- 약초를 가공해 약을 제조
- 게임이 끝나고 성과에 따라 4가지의 엔딩 중 하나가 출력됨
